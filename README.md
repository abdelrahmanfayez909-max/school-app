# نظام إدارة مدرسة المستشار أحمد عصمت

نظام احترافي لإدارة الطلاب يعمل بتقنيات Modern Web (Next.js, Supabase) ويتميز بواجهة عصرية واحترافية.

## المميزات
- **إدارة الطلاب**: إضافة، تعديل، حذف، وبحث متقدم.
- **لوحة تحكم ذكية**: إحصائيات فورية وتنبيهات.
- **تصدير البيانات**: دعم تصدير الجداول إلى Excel.
- **سجل العمليات**: تتبع كامل لكافة الحركات داخل النظام.
- **تصميم عصري**: متوافق مع كافة الأجهزة (Responsive) ودعم كامل للغة العربية (RTL).
- **بدون سيرفر**: يعتمد على Supabase كقاعدة بيانات سحابية (Serverless).

## المتطلبات التقنية
- Node.js 18+
- حساب على [Supabase](https://supabase.com)

## الإعداد والتشغيل

1. **إعداد قاعدة البيانات**:
   - أنشئ مشروعاً جديداً على Supabase.
   - قم بإنشاء جدول باسم `students` بالحقول المطلوبة (الاسم، الرقم القومي، إلخ).
   - قم بإنشاء جدول باسم `logs` لسجل العمليات.

2. **ضبط المتغيرات**:
   - قم بإنشاء ملف `.env.local` في المجلد الرئيسي.
   - أضف القيم التالية:
     ```
     NEXT_PUBLIC_SUPABASE_URL=رابط_مشروعك
     NEXT_PUBLIC_SUPABASE_ANON_KEY=مفتاح_المشروع
     ```

3. **التشغيل محلياً**:
   ```bash
   npm install
   npm run dev
   ```

4. **النشر (Deployment)**:
   - هذا المشروع معد للنشر السحابي (Static Export).
   - يمكنك رفعه مباشرة على Vercel أو Netlify أو GitHub Pages.
   - عند النشر على GitHub Pages، سيتم إنشاء مجلد `out` يحتوي على ملفات الموقع الجاهزة.
